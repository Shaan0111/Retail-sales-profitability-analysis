# Dataset Information

This dataset contains simulated retail sales transactions used for analysis.

## Columns

* Order_ID: Unique transaction ID
* Product: Product name
* Category: Product category
* Price: Unit price
* Quantity: Number of units sold
* Discount: Discount applied (0–30%)
* City: Location of sale
* Revenue: Price × Quantity
* Profit: Revenue adjusted for discount

## Note

The dataset is generated using Python to simulate real-world retail scenarios, including discount impact on profitability.
